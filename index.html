About Me

Hi, I’m Daisylyn Espiritu from Bagacay, Buenavista, Marinduque. I’m currently studying Information Technology at Marinduque State University, specializing in Software Development. With a background in graphic design and web development, I've always been passionate about combining creativity with technology. However, the shift towards software development has presented its own set of challenges. One of the biggest obstacles I’ve faced is adapting from a design-focused mindset to one that is more logical and code-based. In graphic design and web development, I’ve been used to focusing on aesthetics and user experience, but software development requires a different approach—focusing on problem-solving, algorithms, and coding. The learning curve has definitely been steep but also very rewarding. Another challenge has been balancing the visual appeal with the technical functionality in software projects. While web development allows me to create visually engaging interfaces, software development demands that the code behind these designs is efficient and scalable. Debugging and optimizing code has proven to be a whole new skill set that requires a lot of patience and precision. Despite these challenges, I’m committed to expanding my expertise in the IT field. I aim to combine my design skills with my growing software development knowledge, striving to create solutions that are both functional and visually appealing. I look forward to continuing my development as a versatile developer who can bring both creativity and technology together.
